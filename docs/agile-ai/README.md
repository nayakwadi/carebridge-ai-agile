# AI-Powered Agile : Course Demonstration

This folder is the portfolio proof that I can apply the techniques from *AI-Powered Agile: Strategies for Modern Project Managers* (Workbook Chapters 1–3) to a real, regulated product. Each artifact takes a workbook exercise and runs it against **CareBridge** : an AI-assisted patient care-coordination platform built to cut avoidable 30-day readmissions across 3 pilot health systems. Every artifact follows the same disciplined shape: the scenario, the exact structured prompt I used, the AI-generated output, and : most importantly : the **PM judgment** I applied on top of the AI's draft. The point isn't that AI wrote these; it's that a technical PM directed the AI, verified its work, and owns every decision.
